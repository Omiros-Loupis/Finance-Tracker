#  Python Personal Finance Tracker

A robust and secure Personal Finance Tracker built in **Python**. This project utilizes **SQLite** for persistent storage, **Pandas** for powerful data analysis, and **Matplotlib** for insightful visualizations.

| Category | Features |
| **Data Management** | SQLite DB, Add Income/Expenses with Categories, Automatic Date Tracking, Data Export to CSV. |
| **Analysis** | View Transactions in Pandas DataFrames, Financial Summaries (Income, Expenses, Net), Category Breakdown, Monthly Reports. |
| **Visualizations** | Auto-saved PNG charts: Pie Charts for distribution, Bar Charts for comparison. |
| **User Experience** | Interactive Menu, Sample Data Included, Filtering by Month/Year. |
